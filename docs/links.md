### マッピングガイド・ドキュメント

* **<a href="https://bsmg.wiki/" target="_blank">BSMG Wiki</a>**<br>Mappingページには、マッピングの始め方から応用的なマッピング技術まで、非常に多くの情報がまとめられています。マッパー必読。
* **<a href="https://wiki.scoresaber.com/" target="_blank">ScoreSaber Wiki</a>**<br>ScoreSaberのランク基準などが確認できます。
* **<a href="https://beatleader.wiki/en/home" target="_blank">BeatLeader wiki</a>**<br>BeatLeaderのランク基準などが確認できます。
* **<a href="https://docs.google.com/document/d/1NQzXNsQ0VhnCoUBhEb_Qx0gONsqaD8drOMWIXJk79vE/edit?tab=t.0" target="_blank">The Guide to the Group Lighting System</a>**<br>海外のライターたちが作ったV3ライトドキュメント。かなりの情報量です。<br>※コピー制限がかかっているのでPCからのブラウザ翻訳は不可
* **<a href="https://ugecko.github.io/chroodleWeb/" target="_blank">Chroodle</a>**<br>ugecko氏が作ったEnvironment Enhancementのドキュメント。全てのChroma log、Environment情報も載っています。
* **<a href="https://heck.aeroluna.dev/" target="_blank">Heck Wiki</a>**<br>Heck系Mod（Chroma, NoodleExtensions, Vivify）のドキュメントです。
* **<a href="https://chromapper.atlassian.net/wiki/spaces/UG/overview" target="_blank">ChroMapperユーザーガイド</a>**<br>基本的な使い方が載っていますが、内容が一部古い部分があります。
* **<a href="https://beatsaber.com/documentation.html" target="_blank">BeatGames公式ドキュメント</a>**<br>公式譜面クレジットと、公式エディタの解説（情報かなり古め）が載っています。

### お役立ちツール・サイト

* **<a href="https://kivalevan.me/BeatSaber-MapCheck/" target="_blank">Beat Saber Map Check</a>**<br>
    問題のある配置などをチェックできるマップチェッカー
* **<a href="https://kivalevan.me/BeatSaber-MappingUtility/" target="_blank">Beat Saber Mapping Utility</a>**<br>
    細かいツールが色々ある
* **<a href="https://allpoland.github.io/ArcViewer/" target="_blank">ArcViewer (Web版)</a>**<br>
    Web上で使える譜面プレビュー。リプレイも再生できる。
* **<a href="https://github.com/AllPoland/ArcViewer/releases" target="_blank">ArcViewer (デスクトップ版)</a>**<br>
    デスクトップアプリ版。マッピング中のプレビューなどに。
* **<a href="https://converter.stormpacer.xyz/" target="_blank">Map Converter</a>**<br>
    V4からV3への変換ができる。
* **<a href="https://wipbot.com/" target="_blank">wipbotサーバー</a>**<br>
    !wipコマンドで配信者にwip譜面をリクエストしたい時に。
* **<a href="https://github.com/Danielduel/wipbot/releases" target="_blank">wipbot (Mod)</a>**<br>
    wipbotのMod
* **<a href="https://easings.net/" target="_blank">Easing Functions</a>**<br>
    各種イージングを視覚的に確認できるサイト。Chroma、Modchart、V3ライティングのお供に。
* **<a href="https://docs.google.com/document/d/1Zl8jh0djB80o3tbTmGR1MF9gV9pkYPfxqTC4IuRQmd0/edit?usp=sharing" target="_blank">ビーセイゆるふわ用語集</a>**<br>
    hibit氏が作成した、Beat Saber全般のいろいろなことをゆるく解説した用語集。


### その他

* **<a href="https://github.com/Swifter1243/ReMapper" target="_blank">ReMapper</a>**<br>
    Chroma, NoodleExtensions, Vivifyをより高度に使いこなしたい人へ
* **<a href="https://beatmods.com/" target="_blank">BeatMods</a>**<br>
    BSManagerやModAssistantで配信されるModを登録できる（Modの検索・インストールはインストーラー経由を推奨）
* **<a href="https://modelsaber.com/" target="_blank">ModelSaber</a>**<br>
    セイバー、ノーツ、プラットフォーム等のカスタムモデルを登録・検索できる
